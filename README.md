## Тестовое задание:

- Arduino MEGA ATMEGA2560
- К пинам PWM 9, 10, 11 подключен драйвер с мотором шагового двигателя 9 step, 10 dir, 11 ena
- К пину А1 подключен потенциометр
- К пинам PW 2 и 3 подключены кнопки.

# Реализовано: 
- Управление шаговым двигателем, с регулированием скорости от потенциометра в пределах
  от 0 до 2 об/сек
- При нуле снимается ENA
